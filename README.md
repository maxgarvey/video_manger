# video_manger

A localhost web application for serving and playing videos in the browser.

## Overview

`video_manger` is a Go-based web server that runs locally and serves your video files through a browser-based player. Browse your video library and watch them directly in the browser without needing external media players.

## Usage

```
go run main.go
```

Then open your browser to `http://localhost:<port>`.

## Goals

- Serve local video files over HTTP
- Browser-based video playback
- Simple, lightweight, no external dependencies
